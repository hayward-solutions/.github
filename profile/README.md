## Open Source Projects

### [Vincenty](https://github.com/hayward-solutions/vincenty)
A lightweight situational awareness platform for real-time location tracking, group messaging, and map visualisation. A modern alternative to TAK Server, deployable in air-gapped environments.

### [Dispatch](https://github.com/hayward-solutions/dispatch)
A self-service GitHub Actions tool with a web-based interface for repository browsing, environment management, workflow triggering, and deployment history — no CLI or direct repo access required.

### [DocShare](https://github.com/hayward-solutions/docshare)
Self-hosted document sharing with granular permissions, S3 storage, SSO/MFA authentication, team management, and CLI tooling — giving organisations full control over data residency and access.
